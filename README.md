# Belflix

영화 정보와 드라마 정보를 볼 수 있는 React App입니다. 

따로 서버는 구현하지 않고, The Movie DB API를 사용해서 영화, 드라마 정보를 받아왔습니다.

React의 Styled-component 라이브러리를 이용해서 스타일링 했습니다.

## 기능

- Now playing Movies
- Upcoming Movies
- Popular Movies
- Top Rated Shows
- Popular Rated Shows
- Airing Today Shows
- Search Movies and TV shows  
- Movies & TV Shows informations

## 개발 기간

2020.07 ~ 2020.08

## 사용 기술

- React 
- React Hooks
- Styled-component
- Axios
- Netlify
